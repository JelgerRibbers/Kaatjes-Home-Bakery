# Kaatjes Home Bakery website

Statische website voor een ambachtelijke thuisbakkerij met focus op brood.

## Openen
Open `index.html` in je browser.

## Aanpassen
- WhatsApp: zoek in `index.html` naar `31600000000` en vervang dit door het juiste mobiele nummer, zonder `+`.
- E-mail: vervang `info@kaatjeshomebakery.nl` door het echte e-mailadres.
- Prijzen/assortiment: pas de teksten aan in de sectie `id="brood"`.
- Foto's: vervang afbeeldingen in de map `assets` door eigen foto's met dezelfde bestandsnamen.

## Kleuren
- Espresso: `#4B3A2A`
- Korst bruin: `#8B5E34`
- Honing: `#C4935A`
- Linnen: `#DCC9B2`
- Crème: `#F5F1EA`

## Online zetten
Deze site kan naar Netlify, GitHub Pages, Vercel of gewone webhosting worden geüpload.
